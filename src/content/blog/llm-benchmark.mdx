## TL;DR

We benchmarked **7 LLM providers** across 3 model sizes to measure real-world token throughput, time-to-first-token (TTFT), and cost-per-1M tokens.

## Methodology

Each provider was tested with:

- **Prompt:** 500-token system prompt + 100-token user message
- **Max tokens:** 1024
- **Concurrency:** 1 (sequential) and 10 (parallel)
- **Runs:** 50 per configuration
- **Metric:** tokens/second (output), TTFT, total wall-clock time

### Providers Tested

| Provider | Model | Context Window |
|----------|-------|---------------|
| OpenAI | GPT-4o | 128K |
| Anthropic | Claude Sonnet 4 | 200K |
| Google | Gemini 2.5 Pro | 1M |
| Meta | Llama 3.3 70B | 128K |
| Mistral | Mistral Large | 128K |
| DeepSeek | DeepSeek V3 | 128K |
| Groq | Llama 3.3 70B | 128K |

### Cost per 1M Output Tokens

The cost efficiency varies dramatically:

$$
\text{Cost Efficiency} = \frac{\text{Tokens/sec}}{\text{Cost per 1M tokens}} \times 1000
$$

## Key Takeaways

1. **Groq** is the undisputed throughput king — but limited model selection
2. **DeepSeek V3** offers the best price/performance ratio
3. **Claude Sonnet 4** and **GPT-4o** lead in quality-critical tasks
4. **Gemini 2.5 Pro** wins on long-context workloads
